Quick Start

In this guide, we'll go through writing the code to perform a simple matrix-vector product in parallel with MathWorkersJS.

Part 1

Let's get started by writing a super simple HTML web page that displays "hello world":

<!doctype html>
<html>
    <p id="text">hello world</p>
</html>
view page

Easy, right? Let's add a script tag that changes that message:

<!doctype html>
<html>
    <p id="text">hello world</p>

    <script>
        var x = 1 + 2;
        document.getElementById("text").innerHTML = x;
    </script>
</html>
view page

Okay, now we can add in MathWorkersJS. Here, I am just loading the minified verion of MathWorkersJS that is available on this site. You should, of course, host your own copy of MathWorkersJS for your site, and change the src value in the code below correspondingly. We'll make a little Vector object now and print it to the screen: 

<!doctype html>
<html>
    <head>
        <script src="/static/js/mathworkers.min.js"></script>
    </head>

    <body>
        <p id="text">hello world</p>
    </body>

    <script>
        var v = MathWorkers.Vector.fromArray([1, 2, 3, 4]);
        document.getElementById("text").innerHTML = v.toString();
    </script>
</html>
